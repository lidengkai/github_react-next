## 项目部署

**安装依赖**

  npm i

**本地调试**

  npm start

**语法检查**

  npm run tsc(:watch)

**打包环境**

  npm run [dev|test|prod]

  npm run build

**发布目录**

  .next
## eslint

  - js

    - 字符串必须使用单引号
    - 文件末尾使用换行符
    - 使用2个空格缩进
    - 禁用debugger
    - 不使用var
    - 禁用alert confirm prompt
    - 不能对var声明的变量使用delete操作符
    - 不使用分号
    - 禁用不规则的空白
    - 函数调用时，函数名与()之间不能有空格
    - 行末不使用空格

  - jsx
    - 属性中必须使用双引号
    - 省略属性值为true的赋值
    - 多行属性，右括号另起一行
    - 大括号内首尾不使用空格
    - 属性赋值不使用空格
